## Test 3 - Admin create event and login
Viewport: desktop
Description: Verifies admin can create event and login with PIN.

Steps:
1. Navigate to /admin
2. Create event with code (5-10 chars), PIN, duration
3. Expect redirect to dashboard or create success
4. Log out or open new session, go to /admin
5. Login with same event code and PIN

Expected: Create succeeds; login succeeds and shows dashboard.
